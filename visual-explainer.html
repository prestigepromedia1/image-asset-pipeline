AI ENGINE

Three-Tier Identification

Filename Match Cloud Vision OCR Claude Vision _Review/
TIER 1

Filename Matching

Checks filenames against keywords in your category catalog. Instant, free, no API calls needed.

Free Instant No API key required
TIER 2

Cloud Vision OCR

Reads text on product labels and packaging. Maps detected text to categories via your label_text_map.

~$1.50 / 1000 images Google Cloud Vision API Optional
TIER 3

Claude Vision

AI visual analysis identifies products and classifies image style when text isn't readable.

~$0.01-0.03 / image Anthropic API Fallback only

FEATURES

Built For Real Workflows

Local + Google Drive

Use --local for filesystem operations or pass a Drive folder ID. Same commands, same output.

Naming Templates

Control rename output with --template "{category}_{style}_{seq}". Tokens: category, style, seq, date.

Dry Run Mode

Preview organize and rename operations with --dry-run before moving or renaming any files.

Category Catalog

Define categories, keywords, label text maps, and style categories in a simple categories.json.

Batch Processing

Process large folders in batches with --batch N and resume with --start-batch N.

Export Formats

Index outputs CSV, JSON, or both. Include MD5 checksums with --checksums for asset verification.

Quick Start

INSTALL
pip install anthropic Pillow
Core deps only. Cloud Vision and Drive deps are optional.
ORGANIZE
python image_asset_pipeline.py \ organize ./photos \ --local --catalog categories.json
RENAME + INDEX
python image_asset_pipeline.py \ rename ./photos/Organized --local python image_asset_pipeline.py \ index ./photos/Organized --local

Cost-optimized by design. Filename matching is tried first (free). OCR handles labeled products (~$1.50/1000). Claude Vision is only used as a fallback (~$0.01-0.03/image). For a 2,500-image folder, expect under $10 total.
