Image Asset Pipeline

organize → rename → index → dedup

Turn a folder of IMG_4530.jpg files into organized, properly named, searchable assets. Three-tier AI identification sorts images by product, then bulk rename gives them metadata-rich filenames. Works with local folders or Google Drive.

The Pipeline

Step 1
organize
AI sorts images into category folders. Uses filename matching, OCR, then Claude Vision as fallback.
Step 2
rename
Bulk rename with {category}_{style}_{seq} template. Auto-detects product from folder name.
Step 3
index
Export CSV or JSON asset manifest with filename, category, path, Drive link, and optional checksums.
Step 4
dedup
Find duplicates by MD5 checksum. Reports exact dupes and same-name variants across folders.
organize uses three-tier identification

Three-Tier Identification

TIER 1
Filename Matching
Instant pattern matching against your keyword catalog. Checks if the image filename contains known product keywords. Fastest and most reliable for well-named source files.
Free
TIER 2
Cloud Vision OCR
Google Cloud Vision reads text on product labels and packaging. Maps detected label text to categories via label_text_map. Handles most labeled products without AI.
~$1.50 / 1K images
TIER 3
Claude Vision
AI visual analysis for images where text isn't readable. Classifies product + style category. Only fires when OCR fails or isn't enabled. Fallback-only.
~$0.01-0.03 / image

Storage Modes

Local Mode

  • Pass --local flag to any command
  • Source is a filesystem directory path
  • Files moved/renamed directly on disk
  • No Google credentials needed
  • Sort by name or creation date

Google Drive Mode

  • Default mode (no --local flag)
  • Source is a Drive folder ID
  • Files organized via Drive API
  • OAuth2 credentials required
  • Sort by name or createdTime

Naming Templates

Quick Start

# Install pip install anthropic Pillow # Organize a local folder python image_asset_pipeline.py organize ./photos --local --catalog categories.json # Rename organized files python image_asset_pipeline.py rename ./photos/Organized --local # Export asset index python image_asset_pipeline.py index ./photos/Organized --local --format csv # Find duplicates python image_asset_pipeline.py dedup ./photos --local
Google Drive mode. Same commands without --local. Replace the path with a Drive folder ID. Requires credentials.json from Google Cloud Console with Drive API enabled.
